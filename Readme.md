# Node.js Basics

## Available Scripts

Run the following commands using `npm run`:

### File System Operations
- `fs:create` - create a new file
- `fs:copy` - copy folder
- `fs:rename` - rename a file
- `fs:delete` - delete a file
- `fs:list` - list files in a directory
- `fs:read` - read a file

### Command Line Interface
- `cli:env` - parse environment variables
- `cli:args` - parse command line arguments

### Modules & Hashing
- `modules` - rewritten in ECMAScript notation
- `hash` - calculate a hash

### Streams
- `streams:read` - read from a stream
- `streams:write` - write to a stream
- `streams:transform` - reverse text in stream

### Compression
- `zip:compress` - compress a file
- `zip:decompress` - decompress a file

### Worker Threads & Child Processes
- `wt` - run worker threads
- `cp` - run child processes
